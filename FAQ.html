What is Magie Sugary?

Magie Sugary is an upcoming adventure rpg game for PC and Android.

When the game will be released?

As of now it does not have an official release date.

Who is making it?

Everything is being developed solo by Felipe Dounford.

Which game engine are you using?

GameMaker.

Who made the amazing images on characters page?

It was made by the amazing CoMiGo.

Will it have controller support?

Yes, the game will have full controller support and remapping option.

What languages the game will support?

Magie Sugary is planned to be released at least with English and Brazilian Portuguese.

Will this game have multiplayer support?

currently there are no plans for multiplayer.

What is Zulik?

Zulik is the language of Magie Sugary world.

I have an idea for a feature/character how can I talk with you?

It is possible to contact me using the Official Discord Server.

Is this game coming to X platform?

I would like to release on as many different platforms as possible, but some platforms are a lot more difficult to release without a publisher, so for now I can only guarantee that the game will be available on Windows and Android.

What are the system requirements to play Magie Sugary?

Because the game is currently in development, it is hard to say for now.
